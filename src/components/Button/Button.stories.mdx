## Usage

### Sizes

Apart from the standard button, we support slim and large variant buttons.

<Preview>
  <Story name="Sizes">
    <Stack alignment="center" distribution="equalSpacingAround">
      <Button size="slim">Add product</Button>
      <Button size="large">Add product</Button>
    </Stack>
  </Story>
</Preview>

### Styles

An outline button will maintain the appropriate visual weight and won't clash with the
background color. For secondary actions, the outline style is appropriate while the plain
style can be used for tertiary actions.

<Preview>
  <Story name="Styles">
    <div style={{ backgroundColor: "#f6f6f9", padding: "1rem" }}>
      <Stack distribution="equalSpacingAround">
        <ButtonGroup>
          <Button>Invite</Button>
          <Button brandOutline>Invite</Button>
          <Button outline>Invite and add another</Button>
        </ButtonGroup>
        <Button plain>Cancel</Button>
      </Stack>
    </div>
  </Story>
</Preview>

### Disabled

Used to indicate that the button is not clickable. This is used in situations where a
condition has not been fulfilled, for example, when a form has a validation error.

<Preview>
  <Story name="Disabled">
    <Stack distribution="equalSpacingAround">
      <Button disabled>Add product</Button>
      <Button brandOutline disabled>
        Add product
      </Button>
      <Button outline disabled>
        Add product
      </Button>
      <Button plain disabled>
        Add product
      </Button>
    </Stack>
  </Story>
</Preview>

### URL

`Button` accepts a `url` prop when presenting an actionable link (as oppossed to a
navigational link)

<Preview>
  <Story name="URL">
    <Stack distribution="equalSpacingAround">
      <Button url="https://www.google.com">Visit</Button>
      <Button disabled url="https://www.google.com">
        Visit
      </Button>
      <Button brandOutline url="https://www.google.com">
        Visit
      </Button>
      <Button brandOutline disabled url="https://www.google.com">
        Visit
      </Button>
      <Button url="https://www.google.com" outline>
        Visit
      </Button>
      <Button disabled url="https://www.google.com" outline>
        Visit
      </Button>
      <Button url="https://www.google.com" plain>
        Visit
      </Button>
      <Button disabled url="https://www.google.com" plain>
        Visit
      </Button>
    </Stack>
  </Story>
</Preview>

### Download

`Button` accepts a `download` prop for buttons that use the `url` prop to hint to browsers
that it should download the file instead of navigating to it.

<Preview>
  <Story name="Download">
    <Stack distribution="equalSpacingAround">
      <Button url="https://www.google.com" download>
        Download Google homepage
      </Button>
      <Button url="https://www.google.com" download="google.html">
        Customize the filename
      </Button>
      <Button url="https://www.google.com" download={false}>
        Download set to false
      </Button>
      <Button download>No effect on non-URL button</Button>
    </Stack>
  </Story>
</Preview>
